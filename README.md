# UI-Particle-Editor
Editor de Partículas para Otclient Redemption


```bash
# Instalar dependências
- Python 3.10+

py -m pip install -r requirements.txt

# Executar 
py main.py
```
Features:
- Salva em .otps
- Carrega Texturas externas
- Modo Random
- Modo Editor



Como usar?

- Particle Creature:
https://github.com/mehah/otclient/wiki/Tutorial-Attached-Effects#-4--particle

- Particle tile:
https://github.com/mehah/otclient/wiki/Tutorial-Attached-Effects#------42-particle-tile

- Particle Widget:
https://github.com/mehah/otclient/wiki/Tutorial-Attached-Effects#------43-particle-widget



```bash
# No otui:

  UIParticles
    id: particles
    effect: custom_effect
    anchors.fill: parent
    phantom: true	
```
